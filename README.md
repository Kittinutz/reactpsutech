# Basic installation
### Requirement System
- Nodejs ref:https://nodejs.org/en/
- React ref:https://reactjs.org/

```
$ npm install -g create-react-app
$ git clone https://github.com/Kittinutz/reactpsutech.git
$ npm install
$ npm start

```
# Redux devtool configuration
